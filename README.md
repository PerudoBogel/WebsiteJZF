# JZF Website

A simple website with a collapsible top menu, welcoming image, text sections, and photo galleries.

## Features

- Collapsible menu in the top left corner
- Four menu options: two text sections and two photo galleries
- Starting view displays a welcoming image
- Content area below the menu

## Structure

- `index.html`: Main HTML file
- `css/styles.css`: Stylesheet
- `js/script.js`: JavaScript for menu functionality
- `images/`: Directory for images (placeholders included)

## Usage

Open `index.html` in a web browser to view the website.

## Notes

- Placeholder images are referenced but not included. Replace with actual images.
- Menu collapses by default and expands on click.
